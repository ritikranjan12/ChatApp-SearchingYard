
# Chat App

  https://chatapp.coderhunts.com/
```

```

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn
  ```

Start the server

```bash
 yarn dev
```
